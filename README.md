-   考核方式：闭卷考试

1.  签到5%：未到0分、迟到0.6分、请假0.8分、公假和到勤1分。
2.  在线学习环境安装5%：目标3
3.  在线讨论5%：目标4
4.  作业45%（1、4：目标3；2、3：目标1；5、6：目标2；7/8/9：目标4）
5.  在线测试20%：目标1
6.  随堂练习15%：目标2
7.  在线学习次数5%：目标4
-   ## 期末考核（闭卷考试）  
1.  简答题4*5分=20分：目标1和目标2    
2.  辨析题2*5分=10分：目标3和目标4    
3.  程序设计题2*10分=20分：目标1    
4.  综合应用题1-4题_10分+5-6题_5分=50分    
    -   映射类的编写：目标1
    -   接口编写：目标2
